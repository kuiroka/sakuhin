# なつみかん

学習塾で使えるwebアプリケーションを作りました。<br>
現在運用中のため、権利侵害になる可能性のある箇所は消してあります。

# 使用目的

* 書き込みであった単語テストのweb化
* 赤本の補填(現在は問題集の感想のみ)
* 講師間の情報共有のプラットフォームの作成
　

# なぜ自作なのか
企業向けにDXを推し進めている会社は多くあるが、学習塾であるため資金がない。
かつ、塾全体を巻き込んだ企画ではなく、自分が所属している教室単位であるため、外部ではなく自作することにした。
